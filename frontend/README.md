# Naregua App - Frontend

Aplicação React para agendamento de barbearia.

## Tecnologias
- React 18
- Vite
- TailwindCSS
- React Router
- Axios
- React Query

## Como executar

1. Instalar dependências: `npm install`
2. Executar: `npm run dev`
3. Acessar: http://localhost:5173

## Estrutura
- `src/api/` - Chamadas para backend
- `src/components/` - Componentes reutilizáveis
- `src/features/` - Páginas e lógica por feature
- `src/hooks/` - Hooks customizados
- `src/routes/` - Rotas protegidas

## Backend
Certifique-se de que o backend está rodando em http://localhost:3000
